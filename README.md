# Desafio de Front-end feegow

## Project setup

```
git clone https://github.com/joaofds/feegow-app.git
```

## Navegar para o projeto

```
cd feegow-api
```

## Instalar as dependencias com o NPM

```
npm install
```

### Compilar e rodar servidor

```
npm run serve
```
